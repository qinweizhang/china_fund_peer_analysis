# 同业基金分析网页平台

权益竞争对手季度持仓及规模变动分析的网页化呈现。基于 Flask，将 Excel 基础数据经业务逻辑加工后，以可交互网页图表形式展示同业基金公司的业绩、持仓与规模变动。

> 设计语言遵循部门 ADLS v2.1 规范（深海色板 / 完全去背 / 绝对正交 / 等宽数字）。详见 `reference/ADSL.md`。
> 完整需求与口径见 [spec.md](spec.md)。

## 功能模块

| 模块 | 路由 | 内容 |
|------|------|------|
| 01 业绩及规模概况 | `/` | 十五大公司权益规模、业绩排名、规模变动拆分（收益率上涨 / 持营 / 新发） |
| 02 持仓及变动分析 | `/holdings` | 2.1 板块配置（六板块+A股/港股）、2.2 板块配置（A股/港股拆分，各六板块）、2.3 板块变动、2.4 板块变动（A股/港股拆分）、2.5 前十大重仓股、2.6 集中度与仓位四张折线图 |
| 03 规模及变动分析 | `/scale` | 逐季规模与近五季规模变动拆分（按收益率上涨/持营/新发分组，粗线分隔）、图3.1 规模变化区间基金数量+平均收益率（柱+折线）、图3.2 各类型产品规模变动、收益率 vs 最大回撤散点、规模增长/缩减 TOP10、永赢产品专题、产品布局对比 |

## 数据集与上传

平台不止用于分析单期数据，支持多数据集切换与上传，便于持续分析后续季度（2026Q2、Q3……）。

- **数据集下拉**（右上角）：列出 `input_data/`（默认）与 `uploads/`（上传）下所有 `.xlsx`，选择后整站切换到该数据集，所有表格与图表按新数据重算。
- **上传数据**（右上角"＋ 上传数据"按钮）：选择与示范同结构的 Excel（12 个 sheet）即可上传，自动登记为新数据集并切换为激活。文件保存至 `uploads/`，重名自动追加序号。
- 当前激活数据集与"数据截至"时点均在顶栏显示；季度、最新/上一季度均由数据派生，无需改代码即可适配新季度。

## 运行步骤

### 环境依赖

- Python 3.9+
- 依赖：`flask`、`pandas`、`openpyxl`

```bash
# 安装依赖（若 pip 报 SSL 证书错误，追加 --trusted-host pypi.org --trusted-host files.pythonhosted.org）
python3 -m pip install flask pandas openpyxl
```

### 启动

```bash
# 在项目根目录执行
python3 -m app.main
```

浏览器打开 http://127.0.0.1:5000/ 即可查看。三个模块通过顶部导航切换。

## 数据与口径

- 输入：`input_data/竞争对手分析-基础数据20260331.xlsx`，数据截至 2026-03-31，含 12 个 sheet（排名、产品规模、事后分类、持仓明细、行业板块映射、集中度、仓位、收益金额、复权净值、个股收益率、行业收益率）。
- 样本：偏股基金规模前十五大基金公司 + 我司（工银瑞信，排名第 11）。
- 关键口径与“demo 近似”说明见各页面底部脚注及 [spec.md](spec.md) §9。规模、排名、TOP10 产品等数值已与报告 PDF 核对一致；持营/新发拆分、板块变动剔除涨跌幅等口径在第二阶段对齐。
